# UI 디자인 계획 — institution-ui

**문서 버전:** 1.0
**작성일:** 2026-04-15

## 1. 디자인 철학

- **전문적, 신뢰감, 차분함** — 엔터프라이즈 대시보드 스타일 (Jira·Notion 유사)
- 그라데이션·AI 감성 금지. **플랫 단색만** 사용
- 깊이감은 미묘한 그림자와 보더로 표현
- 과도한 애니메이션 지양

## 3. 레이아웃 구조

### 3.1 전체 레이아웃

```
+------------------------------------------------------------+
| NavBar (h-16, 고정)                                        |
+------------------------------------------------------------+
|                                                            |
|   [ max-w-5xl 중앙 정렬 컨텐츠 영역 ]                    |
|                                                            |
|   py-8 px-4                                                |
|                                                            |
+------------------------------------------------------------+
```

### 3.2 레이아웃 규칙

| 항목 | 값 |
|------|-----|
| Body 배경 | `bg-gray-50` |
| NavBar 높이 | `h-16` (64px) |
| NavBar 배경 | `bg-white border-b border-gray-200` |
| 컨텐츠 최대 폭 | `max-w-5xl` (1024px) |
| 컨텐츠 수직 여백 | `py-8` |
| 컨텐츠 수평 여백 | `px-4 sm:px-6 lg:px-8` |
| 로그인/회원가입 | NavBar 없음, 화면 중앙 정렬 |

### 3.3 NavBar

```
+------------------------------------------------------------+
| [로고]  연구기관 관리   기관 목록  기관 유형 |  admin [ADMIN] 로그아웃 |
+------------------------------------------------------------+
```

- 좌측: 로고 + "연구기관 관리" (링크 `/institutions`)
- 중앙: 링크 2개. 현재 경로 하이라이트 (`text-blue-600 border-b-2 border-blue-600`)
- 우측: 사용자명 + role 뱃지 + 로그아웃 버튼
- 모바일 (<640px): 햄버거 메뉴로 축약

## 4. 컴포넌트 디자인

### 4.1 버튼

공통: `h-10 px-4 text-sm font-medium rounded-lg transition-colors`

| 종류 | 클래스 |
|------|--------|
| Primary | `bg-blue-600 text-white hover:bg-blue-700 disabled:bg-gray-300` |
| Secondary | `bg-white text-gray-700 border border-gray-300 hover:bg-gray-50` |
| Danger | `bg-red-600 text-white hover:bg-red-700` |
| Ghost | `text-gray-700 hover:bg-gray-100` |

로딩: `disabled:opacity-50 disabled:cursor-not-allowed` + "처리 중…" 텍스트

### 4.2 폼 입력

```
label  : text-sm font-medium text-gray-700 mb-1
input  : h-10 px-3 border border-gray-300 rounded-lg w-full
         focus: ring-2 ring-blue-600 border-transparent outline-none
error  : border-red-600
helper : text-xs text-gray-500 mt-1
errmsg : text-xs text-red-600 mt-1
```

Select: 동일 스타일 + 드롭다운 아이콘

### 4.3 카드

```
bg-white rounded-lg shadow-sm border border-gray-200 p-6
```

### 4.4 테이블

```
Container : bg-white rounded-lg border border-gray-200 overflow-hidden
Header    : bg-gray-50 text-xs uppercase text-gray-500 font-semibold
            px-4 py-3 text-left border-b border-gray-200
Row       : border-b border-gray-100 hover:bg-gray-50 cursor-pointer
Cell      : px-4 py-3 text-sm text-gray-700
```

빈 상태: `py-12 text-center text-gray-500`

### 4.5 알림 (Alert)

```
성공 : bg-green-50 border border-green-200 text-green-800 p-4 rounded-lg
에러 : bg-red-50 border border-red-200 text-red-800 p-4 rounded-lg
경고 : bg-amber-50 border border-amber-200 text-amber-800 p-4 rounded-lg
```

페이지 상단 고정, 3초 자동 소멸, 닫기(X) 버튼 포함.

### 4.6 모달

```
Overlay : fixed inset-0 bg-black/50 flex items-center justify-center z-50
Panel   : bg-white rounded-lg shadow-xl w-full max-w-md p-6
Title   : text-lg font-semibold text-gray-900 mb-4
Actions : flex justify-end gap-2 mt-6
```

ESC/배경 클릭으로 닫기.

### 4.7 Role 뱃지

```
px-2 py-0.5 text-xs font-medium rounded-full
```

## 5. 화면별 UI

### 5.1 로그인 (`/login`)

```
+---------------------------------------+
|                                       |
|            [ 로고 ]                   |
|         연구기관 관리                 |
|                                       |
|  +---------------------------------+  |
|  |  로그인                         |  |
|  |                                 |  |
|  |  Username                       |  |
|  |  [___________________________]  |  |
|  |                                 |  |
|  |  Password                       |  |
|  |  [___________________________]  |  |
|  |                                 |  |
|  |  [       로그인         ]       |  |
|  |                                 |  |
|  |  계정이 없으신가요? 회원가입   |  |
|  +---------------------------------+  |
+---------------------------------------+
```

| 요소 | 스타일 |
|------|--------|
| 컨테이너 | `min-h-screen flex items-center justify-center bg-gray-50` |
| 카드 | `w-full max-w-md` + 공통 카드 스타일 |
| 로고 영역 | 카드 상단 외부 중앙 |

### 5.2 회원가입 (`/signup`)

로그인과 동일 레이아웃. Role은 라디오 버튼 (USER / ADMIN).

### 5.3 기관 목록 (`/institutions`)

```
+------------------------------------------------------------------+
|  NavBar                                                           |
+------------------------------------------------------------------+
|  기관 목록                                       [+ 기관 등록]  |
|                                                                  |
|  +--------------------------------------------------------------+|
|  | 기관명 | 코드 | 유형 | 대표자 | 설립일                    ||
|  |--------|------|------|--------|---------------------------||
|  | ...    | ...  | ...  | ...    | ...                        ||
|  +--------------------------------------------------------------+|
+------------------------------------------------------------------+
```

- 페이지 제목: `text-2xl font-semibold`
- ADMIN 등록 버튼: 우측 상단 Primary 버튼
- 테이블 행 호버/클릭 → 상세 이동
- 빈 상태: "등록된 기관이 없습니다"

### 5.4 기관 상세 (`/institutions/[id]`)

```
+------------------------------------------------------------------+
|  ← 목록으로                                   [수정] [삭제]      |
|                                                                  |
|  +--------------------------------------------------------------+|
|  |  한국과학기술원 (KAIST)                                      ||
|  |                                                              ||
|  |  유형        대학                                            ||
|  |  주소        대전 유성구 ...                                 ||
|  |  ...                                                         ||
|  |                                                              ||
|  |  ---                                                         ||
|  |  등록일  2026-04-15 10:30  (font-mono)                       ||
|  |  수정일  2026-04-15 10:30                                    ||
|  +--------------------------------------------------------------+|
+------------------------------------------------------------------+
```

- 제목: `text-2xl font-semibold`
- 라벨-값 그리드: 2열 (`grid grid-cols-[160px_1fr] gap-y-3`)
- ADMIN만 수정/삭제 버튼 (수정: Secondary, 삭제: Danger)

### 5.5 기관 등록 (`/institutions/new`)

```
+------------------------------------------------------------------+
|  ← 목록으로                                                      |
|                                                                  |
|  기관 등록                                                       |
|                                                                  |
|  +--------------------------------------------------------------+|
|  | [ 폼 필드 세로 스택 ]                                        ||
|  |                                                              ||
|  | 기관명 *                                                     ||
|  | [_______________________________]                            ||
|  |                                                              ||
|  | 코드 *  (A-Z, 0-9, _, -)                                     ||
|  | [_______________________________]                            ||
|  |                                                              ||
|  | 유형 *                                                       ||
|  | [ 대학                        ▼ ]                            ||
|  |                                                              ||
|  | 주소 *                                                       ||
|  | [_______________________________]                            ||
|  |                                                              ||
|  | 전화 / 이메일 / 홈페이지 / 대표자 / 설립일                   ||
|  |                                                              ||
|  |                              [ 취소 ]  [ 등록 ]              ||
|  +--------------------------------------------------------------+|
+------------------------------------------------------------------+
```

- 폼 필드 간격: `space-y-4`
- 필수 필드: 라벨에 빨간 `*`
- 액션 버튼: 우측 정렬, 취소(Secondary)·등록(Primary)

### 5.6 기관 수정 (`/institutions/[id]/edit`)

등록과 동일. 초기값 사전 채움.

### 5.7 기관 삭제 확인 모달

```
  +----------------------------------------+
  |  기관 삭제 확인                    [X] |
  |                                        |
  |  삭제를 진행하려면 아래 값을 정확히    |
  |  입력해 주세요.                        |
  |                                        |
  |  기관 코드                             |
  |  [__________________________]          |
  |                                        |
  |  기관명                                |
  |  [__________________________]          |
  |                                        |
  |  ⚠ 삭제된 기관은 복구할 수 없습니다.   |
  |                                        |
  |              [ 취소 ]  [ 삭제 ]        |
  +----------------------------------------+
```

- 삭제 버튼: Danger
- 경고 문구: `text-amber-700 text-sm`
- 불일치 시 입력 하단 에러 메시지

### 5.8 유형 목록 (`/institution-types`)

기관 목록과 동일 패턴. 컬럼: 코드, 명칭, 설명, 정렬순서, 활성여부(뱃지).

### 5.9 유형 상세/등록/수정

기관과 동일 패턴. 필드는 code, name, description, sortOrder, active.

### 5.10 유형 비활성화 모달

삭제 모달과 유사하나 재입력 요구 없음. 단순 확인만.

## 8. 반응형 브레이크포인트

| 이름 | 폭 | 동작 |
|------|-----|------|
| 기본 (모바일) | < 640px | 단일 컬럼 스택, NavBar 햄버거 메뉴, 테이블 카드 뷰 |
| `sm` | ≥ 640px | 기본 폼 레이아웃 |
| `md` | ≥ 768px | NavBar 전체 표시 |
| `lg` | ≥ 1024px | 전체 레이아웃, 테이블 표시 |
